## Règle d'or

<!-- commune: all -->

- Le **peuple** élit TOUS les législatifs (3 niveaux) + les exécutifs cantonal et communal
- **EXCEPTION :** le Conseil fédéral et le Tribunal fédéral sont élus par l'**Assemblée fédérale**, PAS par le peuple
- **Tous les exécutifs = 7 membres. Toujours.**

## Parlement fédéral = Assemblée fédérale

<!-- commune: all -->

Deux noms, même chose. Composé de deux chambres :

- **Conseil national** = 200 membres = chambre du **peuple** (national → nation → peuple)
- **Conseil des États** = 46 membres = chambre des **cantons** (états → états → cantons)
- **Palais fédéral** (Berne) = où siège le Parlement

Ensemble, ils élisent :
- le **Conseil fédéral** (gouvernement, 7 membres)
- le **Tribunal fédéral** (justice, Lausanne)

## Rôles

<!-- commune: all -->

- Législatif = **élabore** les lois
- Exécutif = **applique** les lois et gouverne
- Judiciaire = s'assure que les lois soient bien appliquées
- Ministère public = conduit les **investigations** et défend la **société** devant les tribunaux
- Chancelier d'État = **porte-parole** du gouvernement VD
- Droits fondamentaux → dans la **Constitution** (pas Code civil, pas Code pénal)
- Référendum = consultation populaire sur une loi
- Initiative cantonale = **12'000 signatures en 4 mois**
- Unités administratives VD = **districts, arrondissements, communes**
- Vote dès **18 ans** / étrangers votent au niveau **communal uniquement**
- PS (Parti socialiste) = **gauche**
- DSAS = Département de la **santé** et de l'**action sociale**
